Онлайн кітап • Web жобалау

Қош келдіңіз!

Бұл онлайн кітапта сіз Web-жобалаудың негізгі ұғымдарын түсінікті тілмен меңгересіз: веб-бет құрылымы, стиль беру, және интерактивті элементтер жасау. Әр тарауда теория, мысал, терминдер және тест тапсырмалары бар.

Ереже
HTML

Тегтерді дұрыс жабу және мағыналы (семантикалық) тегтерді қолдану бетті түсінікті етеді.

Кеңес
CSS

Дизайн жинақы болсын: 2–3 негізгі түс, жеткілікті аралық, оқуға ыңғайлы қаріп өлшемі.

Тәжірибе
JS

Интерактивті шағын қадаммен бастаңыз: батырмаға click оқиғасын қосып, мәтінді өзгертіңіз.

Оқушылар нені үйренеді?
HTML арқылы веб-беттің құрылымын құру; CSS арқылы дизайн мен безендіру жасау; JavaScript арқылы батырма, форма, тексеру сияқты интерактив қосу; қарапайым веб-жобаны жинақтап құрастыру.
Сәтті оқу тілейміз! Әр сабақты ретімен оқып, мысалдарды қайталап, соңында тест арқылы өзіңізді тексеріңіз. Тренажерде код жазып тәжірибе жасауды ұмытпаңыз 🙂

Кіріспе

Web-жобалау — интернетте жұмыс істейтін веб-беттерді құрастыру өнері мен технологиясы. Веб-бет тек мәтін емес: сурет, батырма, кесте, форма, мәзір, анимация және интерактив арқылы пайдаланушыға ыңғайлы орта қалыптастырады. Бұл оқулықта сіз 3 негізгі құралды біріктіріп қолданасыз: HTML (құрылым), CSS (стиль), JavaScript (қимыл/логика).
