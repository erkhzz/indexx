Терминдер мен анықтамалар

Төмендегі сөздер теориялық бөлімде жиі кездеседі. Әр термин қысқа әрі нақты түсіндірілген.

Ереже
HTML

Тегтерді дұрыс жабу және мағыналы (семантикалық) тегтерді қолдану бетті түсінікті етеді.

Кеңес
CSS

Дизайн жинақы болсын: 2–3 негізгі түс, жеткілікті аралық, оқуға ыңғайлы қаріп өлшемі.

Тәжірибе
JS

Интерактивті шағын қадаммен бастаңыз: батырмаға click оқиғасын қосып, мәтінді өзгертіңіз.

HTML
HTML — веб-беттің құрылымын сипаттайтын белгілеу тілі (тақырып, мәтін, сурет, сілтеме, тізім т.б.).
CSS
CSS — веб-беттің дизайнын және сыртқы көрінісін басқаруға арналған стильдер тілі (түстер, өлшем, орналасу).
JavaScript
JavaScript — веб-бетке интерактив қосатын бағдарламалау тілі (батырма әрекеті, тексеру, анимация, логика).
Тег
Тег — HTML-дегі элементті белгілейтін жазу (мысалы: <p>, <h1>).
Элемент
Элемент — ашылу/жабылу тегтері мен олардың мазмұны (мысалы: <p>Мәтін</p>).
Атрибут
Атрибут — тегке қосымша ақпарат беретін параметр (мысалы: href, src, class).
Гиперсілтеме (Link)
Басқа бетке/сайтқа өтуді қамтамасыз ететін сілтеме. HTML-де <a> тегімен жасалады.
URL
URL — интернеттегі ресурс мекенжайы (мысалы, сайт бетінің сілтемесі).
Браузер
Веб-беттерді ашып көрсететін бағдарлама (Chrome, Edge, Yandex, Safari т.б.).
DOM
DOM (Document Object Model) — браузердегі HTML құжатының “ағаш” түріндегі моделі. JavaScript DOM арқылы элементтерді өзгерте алады.
Селектор
Селектор — CSS-те қай элементке стиль қолданылатынын көрсететін ереже (мысалы: p, .box, #title).
Box Model
Элементтің құрылымы: content, padding, border, margin.
Flexbox
CSS арқылы элементтерді бір өлшемді бағытта (қатар/баған) ыңғайлы орналастыру тәсілі.
Grid
CSS арқылы элементтерді тор (кесте емес) принципімен қатар/бағанға бөліп орналастыру жүйесі.
Оқиға (Event)
Пайдаланушы әрекетіне байланысты болатын жағдай (мысалы: click, input, submit). JavaScript оқиғамен жұмыс істейді.
Функция
Белгілі бір жұмысты орындайтын қайта қолданылатын код бөлігі.
Семантика (Semantic HTML)
Семантика — HTML элементтерінің мағынасы. Дұрыс семантикалық тегтер қолжетімділік пен іздеу жүйелеріне пайдалы.
Responsive дизайн
Responsive дизайн — бет орналасуының экран өлшеміне бейімделуі (телефон, планшет, компьютер).
Media query
Media query — CSS-те экран өлшеміне/құрылғыға қарай стиль қолдануға мүмкіндік беретін шарт.
Қолжетімділік (Accessibility)
Қолжетімділік — сайтты мүмкіндігі әртүрлі адамдарға ыңғайлы ету қағидалары (мәтін баламасы, дұрыс құрылым, клавиатурамен басқару).
Viewport
Viewport — браузер терезесіндегі көрінетін аймақ. Мобильді құрылғыда дұрыс масштаб үшін <meta name="viewport" ...> қолданылады.
HTTP
HTTP — браузер мен сервердің дерек алмасу протоколы (сұраныс–жауап қағидасы).
HTTPS
HTTPS — HTTP-тің қауіпсіз нұсқасы; дерек тасымалдау кезінде шифрлау қолданылады.
DNS
DNS — домен атын (example.com) IP-мекенжайға айналдыратын жүйе.
HTTP статус-коды
Статус-код — сервер жауабының нәтижесін көрсететін сан (мысалы: 200, 404, 500).
Каскад (Cascade)
Каскад — CSS ережелерінің бір-бірін басып озу реті: маңыздысы, нақтылығы, жазылу тәртібі ескеріледі.
Specificity (Нақтылық)
Specificity — селекторлардың “нақтылық” деңгейі. Нақтылығы жоғары ереже басым болады.
Мұрагерлік (Inheritance)
Мұрагерлік — кейбір CSS қасиеттері ата-элементтен балаға беріледі (мысалы: color, font-family).
Keyframes
Keyframes — CSS анимациясының кадрларын сипаттайтын ереже: @keyframes атау {...}.
Event (Оқиға)
Оқиға — пайдаланушы әрекеті немесе жүйелік құбылыс (click, input, submit).
Console
Console — браузердегі құрал. JS қателерін көруге және console.log() арқылы тексеруге болады.
