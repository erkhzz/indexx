Код жазыңыз, бірден көріңіз

Бұл тренажерде сіз үш бөлікті бірге қолданасыз: HTML (құрылым), CSS (дизайн) және JavaScript (интерактив). Іске қосу үшін Ctrl + Enter пернелерін басуға болады.

Ереже
HTML

Тегтерді дұрыс жабу және мағыналы (семантикалық) тегтерді қолдану бетті түсінікті етеді.

Кеңес
CSS

Дизайн жинақы болсын: 2–3 негізгі түс, жеткілікті аралық, оқуға ыңғайлы қаріп өлшемі.

Тәжірибе
JS

Интерактивті шағын қадаммен бастаңыз: батырмаға click оқиғасын қосып, мәтінді өзгертіңіз.

🧱 HTML

Тақырып, абзац, тізім, сурет, батырма сияқты элементтер арқылы беттің құрылымын құрасыз.

🎨 CSS

Түс, өлшем, арақашықтық, орналасу және шағын анимациялармен дизайнын реттейсіз.

⚙️ JavaScript

Батырма басу, санау, мәтінді өзгерту, шарт қою сияқты әрекеттер арқылы бетті “жандандырасыз”.

Кеңес: Алдымен “Үлгі код” батырмасымен бастап, кейін өз идеяңыз бойынша өзгертіңіз. Қателессеңіз, кодты бөліп-бөліп тексеріңіз: алдымен HTML, содан соң CSS, ең соңында JS.
Код
Редактор
Ctrl+Enter • “Іске қосу” батырмасы
HTML: құрылым. Мысалы: <h1>Тақырып</h1>
CSS: дизайн. Мысалы: h1 { color: #1e88e5; }
JavaScript: интерактив. Мысалы: btn.addEventListener('click', ...)
Жұмыс тәртібі: 1) Үлгі кодты жүктеу 2) Өзгерту 3) Іске қосу 4) Нәтижені тексеру. Егер нәтиже шықпаса, алдымен HTML тегтерінің дұрыс жабылғанын қараңыз.
